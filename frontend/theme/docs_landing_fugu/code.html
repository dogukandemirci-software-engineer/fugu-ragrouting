Quickstart Guide

Get up and running with RAGFlow Pro in minutes. Learn how to initialize the client, connect your first data source, and execute a basic semantic search query.

bolt

Before you begin

Ensure you have an active API key from the RAGFlow developer dashboard. You will need this to authenticate your client requests.

1. Installation

Install the official SDK using your preferred package manager. The SDK is currently available for Node.js and Python.

Terminal
# Using npm
npm install @ragflow/client

# Using yarn
yarn add @ragflow/client

2. Initialize Client

Initialize the client with your project API key. We recommend storing this as an environment variable.

index.ts main.py
import { RAGClient } from '@ragflow/client';

// Initialize with your API key
const client = new RAGClient({
  apiKey: process.env.RAGFLOW_API_KEY,
  environment: 'production'
});
